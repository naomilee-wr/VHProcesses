Standard Criteria for Product Requirements¶


This standard identifies the set of activities to be fulfilled by the Product Manager before involving the Engineering/Product Architect.


  • All mandatory fields completed for each product requirement in the Requirements Management system (e.g., Jira Agile) according to the Feature/Epic Template Guideline - Jira Agile

    • Product Requirement Manadatory fields:

      • Investment Category

      • Sponsor

      • Description

      • Customer Outcome

      • Documentation

      • Rationale

      • Acceptance Criteria

  • Detail use cases created for requirements to help both Engineering team - Development and Testing.
